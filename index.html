🔥 Connect to Firebase Cloud — One Time Setup

Your data will be stored securely on Google Firebase. Access from phone, laptop, anywhere. Free forever.

1Create free Firebase project

Go to console.firebase.google.com → Sign in with Google account → Click "Add project"

Name: gold-ledger → Disable Analytics → Click Create project

2Create Firestore database

Left menu → Build → Firestore Database → Create database

Select "Start in test mode" → Region: asia-south1 (India) → Enable

3Get your config

Click ⚙️ gear → Project settings → scroll to "Your apps" → click Web (</>)

Register with any name → copy the firebaseConfig object values below

apiKey: "AIzaSyXXXXXXXXXXXXXXXX"
authDomain: "gold-ledger-xxxxx.firebaseapp.com"
projectId: "gold-ledger-xxxxx"
storageBucket: "gold-ledger-xxxxx.appspot.com"
messagingSenderId: "123456789012"
appId: "1:123456789:web:abcdef123456"
4Paste your values here
